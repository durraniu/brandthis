# Get image URL(s) and/or path(s)

Get image URL(s) and/or path(s)

## Usage

``` r
get_image_paths(img = NULL, browse = FALSE, type = c("personal", "company"))
```

## Arguments

- img:

  Image URL(s) and/or path(s).

- browse:

  Whether to browse for image(s) in the file system. Default is `FALSE`.

- type:

  Brand type. `"personal"` or `"company"`.

## Value

Image URL(s) and/or path(s) as a character vector.
